# WEB-Quiz

A web-based interactive quiz application designed to test users on core web development topics including HTML, CSS, and JavaScript. The project features an organized interface to review and practice multiple-choice questions (MCQs) and true/false exercises dynamically.

## Live Demo
You can view and test the live application directly in your browser here:
[https://mido266.github.io/WEB-Quiz/](https://mido266.github.io/WEB-Quiz/)

---

## Features
* **Diverse Question Types:** Includes structured Multiple Choice Questions (MCQs) and True/False questions for comprehensive practice.
* **Responsive Layout:** Optimized to render cleanly across desktop, tablet, and mobile displays.
* **Interactive Evaluation:** Utilizes client-side scripting to process answers and calculate scores instantaneously.

---

## Built With
* **HTML5:** Structures the quiz layout, questions, and form elements.
* **CSS3:** Handles the visual presentation, styling, and UI component alignment.
* **JavaScript:** Manages the quiz logic, score calculation, and user interactions.
